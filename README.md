# 🍕 Pizza Sales Performance Analysis | Microsoft Excel 

## 📌 Project Overview

This project analyzes pizza sales data to understand sales performance, customer ordering patterns, and product trends. Using Microsoft Excel, I built an interactive dashboard to visualize key performance indicators (KPIs) and generate business insights that support data-driven decision-making.

---

## 🎯 Business Problem

Businesses need to understand their sales performance to make informed decisions about inventory, staffing, and product offerings. This project aims to answer key business questions such as:

- Which pizzas generate the highest revenue?
- Which days and hours have the highest sales?
- Which pizza categories and sizes are most popular?
- Which products perform well and which require attention?

---

## ❓ Questions Answered

- What is the total revenue generated?
- What is the average order value?
- How many orders and pizzas were sold?
- Which days have the highest number of orders?
- During which hours are customer orders highest?
- Which pizza categories and sizes contribute the most to sales?
- Which pizzas are the top and bottom performers?
- What business actions can improve overall sales performance?

## 🧹 Data Preparation

The dataset was prepared before analysis by:

- Validating the dataset for consistency
- Standardizing categorical values (e.g., replacing **M** with **Medium** and **L** with **Large**)
- Formatting data for accurate reporting
- Creating calculated fields required for KPI calculations
- Organizing the dataset for Pivot Table analysis

---

## 📌 Key Insights

- Weekend sales were higher than weekdays, indicating increased customer demand.
- Customer orders peaked during lunch and evening hours.
- Large-sized pizzas contributed the highest share of sales.
- The Classic pizza category generated significant revenue.
- Identifying top and bottom-selling pizzas helps support menu optimization and promotional planning.
